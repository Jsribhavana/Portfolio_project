AWS VPC Network Setup and Documentation Tool

The AWS VPC Network Setup and Documentation Tool was developed to simplify the visualization and documentation of cloud network infrastructure. This project helps users understand AWS Virtual Private Cloud (VPC) architecture by mapping key networking components and presenting them in a structured format.

The tool improves infrastructure visibility, making it easier for engineering teams to analyze network configurations, troubleshoot connectivity issues, and maintain accurate cloud documentation. Through this project, I gained practical experience with AWS networking concepts and cloud infrastructure design.

  • Improved understanding of AWS cloud networking

Madhumeha – AI-Based Diabetes Prediction System

Madhumeha is an AI-powered healthcare application designed to predict the risk of diabetes using patient health parameters. The system utilizes machine learning algorithms such as LightGBM and XGBoost to analyze clinical data and generate accurate predictions.

The project includes data preprocessing, feature selection, and model training to improve prediction performance and reliability. By applying advanced machine learning techniques, the system provides efficient classification while supporting early risk assessment.

  • Efficient and reliable prediction workflow
